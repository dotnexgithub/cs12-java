## Primitives and Referenced Data Types
- Referenced data types have uppercase first letters
    - Strings are not primitive
- Booleans can contain expressions eval. to T/F
- Default values
    - Primitives: usually zero
    - Reference types: null
- Formula for assigning value to ref types
    - [DataType] [varName] = new DataType(); <-- called a constructor
